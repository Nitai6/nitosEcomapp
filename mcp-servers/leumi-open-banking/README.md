# Bank Leumi Open Banking MCP

PSD2-style OAuth2 wrapper around https://www.leumiopenbanking.co.il/apis used exclusively by `finance-il`.

## Setup

1. Register paidads as a TPP at https://www.leumiopenbanking.co.il/apis (sandbox first, prod requires KYC + IL business docs).
2. From their developer portal copy:
   - Client ID → `LEUMI_OPEN_BANKING_CLIENT_ID`
   - Client Secret → `LEUMI_OPEN_BANKING_CLIENT_SECRET`
   - Redirect URI (you set this) → `LEUMI_OPEN_BANKING_REDIRECT_URI` (default `http://localhost:3000/leumi/callback`)
3. `LEUMI_ENV=sandbox` (or `prod` once approved).
4. `cd mcp-servers/leumi-open-banking && npm install`

## First-time auth (one-shot, owner runs once)

In Claude Code, after MCP is loaded:
1. Call `leumi_oauth_url` → opens consent URL
2. Sign into Leumi business banking → consent → redirect URL contains `?code=XXXX`
3. Call `leumi_exchange_code` with that `code` → tokens saved to `state/secrets/.leumi-tokens.json`
4. Done. The wrapper auto-refreshes the access token (Leumi tokens are 60 min lived; refresh tokens last ~90 days).

## Safety guards

- `leumi_initiate_payment` requires both `confirm: true` AND `idempotency_key` (UUID generated by the calling agent). Without these the call rejects.
- Beneficiaries (רשות המסים / ביטוח לאומי / מע"מ accounts) must be pre-registered in your Leumi business beneficiary list — the API will not let you pay an unregistered account.
- `finance-il` SKILL Phase C.3 enforces additional gates (cash-position check, sufficient ledger lead time, etc.).

## Tools

| Tool | Used by `finance-il` Phase |
|---|---|
| `leumi_oauth_url`, `leumi_exchange_code` | one-time auth |
| `leumi_get_accounts`, `leumi_get_balances` | A (daily reconciliation) |
| `leumi_get_transactions` | A |
| `leumi_initiate_payment` | C.3 (1st-of-month tax + BL deposits) |
| `leumi_get_payment_status` | C.3 follow-up |

## When prod credentials aren't ready

Set `LEUMI_ENV=sandbox` and call against the sandbox endpoint — `finance-il` will run end-to-end in dry-mode (read-only data + simulated payment receipts) until you flip to `prod`.
