# Segmentation — Key Segments

| Segment | Definition | Frequency |
|---|---|---|
| **X-Day Engaged** | Subscribed + opened/clicked in past 30–120d + 0 bounces. Start 30, expand to 60/90/120 as opens improve | 3×/week (base) |
| **Unengaged** | Subscribed but inactive | 1×/month re-engagement |
| **Collection/Category Interest** | Viewed/carted/purchased specific collection | 1–2× extra/month |
| **Cross-Sell** | Purchased item X | targeted Y promo |
| **Received-but-not-opened** | Did not open last campaign | re-send with new subject |

> Gruns emails are the design reference (we copy them).
