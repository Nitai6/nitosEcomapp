# Meta Ads Naming Convention

## Format

Each level uses `–` as a delimiter. Every name starts with the brand tag.

### Campaign

```
[Brand] – Product – Budget Type – Phase – Objective
```

**Example:** `[XX] – [Your Product] – ABO – Testing – Purchase`

### Ad Set

```
[Brand] – Product – Targeting Type (Detail) – Countries – Angle – $Budget/day
```

**Example:** `[XX] – [Your Product] – Open (Broad) – US/CA/UK/AU/NZ/EU – Meme-Humor – $15/day`

### Ad

Keep it short and descriptive — just what the ad is.

```
Creative Name – Copy Variant
```

**Examples:**
- `Drake – Med`
- `Distracted-Boyfriend – Short`
- `Heist-Crew-v2 – Long`
- `Before-After – Med`

---

## Field Values

### Brand Tag
| Tag | Brand |
|-----|-------|
| [XX] | [Your Brand/Product abbreviation] |

### Product
- `[Your Product]` — what you're advertising

### Budget Type (Campaign level)
- `ABO` — Ad Set Budget Optimization
- `CBO` — Campaign Budget Optimization
- `Adv+` — Advantage+ Shopping/Campaign

### Phase (Campaign level)
- `Testing` — New creatives/angles, learning phase
- `Scaling` — Proven winners, increasing spend
- `Retargeting` — Warm audiences

### Objective (Campaign level)
- `Purchase` — Optimizing for purchases
- `LPV` — Landing page views
- `Traffic` — Link clicks
- `Leads` — Lead generation

### Targeting Type (Ad Set level)
- `Open (Broad)` — No interest/behavior targeting, let pixel learn
- `Open (Advantage+)` — Advantage+ audience
- `Interest ([Niche])` — Interest-based targeting
- `LAL (Purchase 1%)` — 1% Lookalike from purchasers
- `LAL (LPV 1%)` — 1% Lookalike from landing page viewers
- `RT (Web 30d)` — Retargeting website visitors, 30-day window
- `RT (IG 90d)` — Retargeting Instagram engagers, 90-day window
- `RT (VV 75%)` — Retargeting 75% video viewers

### Countries (Ad Set level)
- `US/CA/UK/AU/NZ/EU` — Full Tier 1 (current default, EU = DE/FR/NL/SE/NO/DK/FI/CH/AT/BE/IE)
- `US/CA` — North America only
- `US` — US only
- `UK/EU` — Europe only
- `APAC` — HK/JP

### Angle (Ad Set level)
- `Meme-Humor` — Meme-format scroll-stoppers
- `Transformation` — Before/after, evolution
- `Social-Proof` — Testimonials, reviews, community screenshots
- `Aspirational` — Vision, identity, future state
- `Authority` — Credibility, expertise
- `Value-Stack` — What you get breakdown
- `Pain-Point` — Problem-focused

### Creative Name (Ad level)
Use the descriptive name of the creative (e.g., `Drake`, `Distracted-Boyfriend`, `UNO-Draw-25`, `Heist-Crew-v2`)

### Copy Variant (Ad level)
- `Short` — Under 125 chars
- `Med` — 2-3 sentences
- `Long` — 3-4 sentences with line breaks
