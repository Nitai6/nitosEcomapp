# Theme Page Operator (7-platform faceless niche playbook)

Builds **faceless theme pages** — niche-locked aggregator accounts that grow fast on sourced/re-edited content, monetize through affiliate/product/leads/sponsorships, and operate as portfolio assets.

**This is NOT a personal brand skill.** No founder voice, no selfies, no thought-leadership. Pure niche entertainment / utility / curiosity. Different job from `social` (personal Reels) and `social-content` (personal/company strategy).

## Core principle

> A theme page is a **distribution business**, not a content business. The win comes from packaging — finding/creating content the algorithm wants, posted at the right cadence, in the right format, in a niche narrow enough that the algorithm can categorize you in 20 posts.

## Hard rules (apply to every theme page)

1. **Narrow niche or die.** "Fitness" loses. "Calisthenics for skinny guys over 30" wins. Algorithm needs ~20 posts to classify you — wide niche = unclassified = no reach.
2. **Faceless** unless niche specifically demands face (e.g. fitness transformation pages). Reduces production cost 10×, makes outsourcing/scaling possible, removes founder bottleneck.
3. **One niche per page.** Never mix. Open a second page for a second niche.
4. **Cadence > quality** in growth phase (first 90 days). 3-5 posts/day on short-form platforms beats 1 perfect post/week.
5. **Sourcing is a system, not a scramble.** Pre-built content pipeline (creators to scrape, licensing approach, re-edit template) before posting.
6. **Hook in 0.5s on TT/IG/YT Shorts.** First frame must stop the scroll. No "hi guys" intros — ever.
7. **Monetize after the audience exists.** Followers before offers. Most pages should hit threshold (per-platform, see below) before any link in bio.
8. **Repurpose ruthlessly.** One content unit → 7 platforms. Edit-once-publish-everywhere via Metricool/Buffer/Publer.

## Platform playbooks

Each platform has its own algorithm, format, cadence, and monetization gate. Treat them as separate businesses sharing a content engine.

### YouTube (Long + Shorts)

**The opportunity**: highest revenue per view of any platform (RPM $2-30 depending on niche). Shorts feed long-form views via "shorts shelf". Long-form = compounding asset (videos earn for years).

**Niche selection**:
- High-RPM niches: personal finance, business, tech reviews, real estate, insurance, software, B2B
- Low-RPM but easy growth: gaming, reaction, compilation, ASMR, pets
- Pick by goal: ad revenue → high-RPM. Audience for product → match product.

**Format mix**:
- **Shorts**: 1-3/day, 15-60s, hook in 0.5s, vertical 9:16. Source: TikTok winners (re-edit), public-domain footage, AI-generated.
- **Long-form**: 1-2/week, 8-15 min for ad revenue (need 8min+ for mid-rolls), thumbnail + title is 80% of CTR
- **Faceless long-form formats**: top-10 lists, documentary-style narration over stock footage, AI-narrated explainers, compilations with commentary, reaction-style with screen recording

**Growth mechanic**:
- Shorts surface to non-subs aggressively — best for raw growth
- Long-form drives revenue + subscriber loyalty
- Strategy: Shorts to grow, long-form to monetize. Shorts hook → "full breakdown on the channel" CTA → long-form.

**Title + thumbnail rules** (long-form):
- Title: curiosity gap or stark claim ("I tested every X for 30 days", "Why X is dead", "The truth about Y")
- Thumbnail: 1-2 elements max, high contrast, face/object close-up, NO mini-text dumps. Use red/yellow accents.
- A/B test thumbnails via TubeBuddy / VidIQ / YouTube native A/B test

**Monetization gates**:
- YPP (ads): 1,000 subs + 4,000 watch hours OR 1,000 subs + 10M Shorts views in 90d
- Shorts Fund replaced by ad revenue share — Shorts earn after YPP
- Sponsorships: viable from 10k-50k subs depending on niche
- Channel sale: faceless YouTube channels sell for 24-48× monthly profit on Flippa/Empire Flippers

**Tools**: VidIQ / TubeBuddy (SEO + ideation), Pictory / Invideo (AI video), ElevenLabs (voiceover), Epidemic Sound (music license)

### Instagram (Reels + Carousels + Stories)

**The opportunity**: highest engagement of any platform when niche is right. Reels reach > anything else on IG. Carousels = save machine = secondary algorithm signal.

**Niche selection**:
- Strong on IG: aesthetic niches (interiors, fashion, food, travel, fitness, pets, art), self-improvement, motivational, business-tactical
- Weak on IG: hyper-tactical B2B, news, raw entertainment (TT wins those)

**Format mix**:
- **Reels**: 2-3/day in growth phase. 7-15s outperforms longer. 9:16. First frame static + bold text overlay.
- **Carousels**: 3-5/week. 8-10 slides. Slide 1 = hook (same energy as Reel hook). Slides 2-9 = value. Slide 10 = soft CTA + save prompt.
- **Stories**: 5-10/day. Polls, quizzes, behind-account, sliders. Drives saves on grid posts.

**Growth mechanic**:
- Reels = reach, Carousels = saves (algo signal), Stories = warm audience retention
- Cross-share Reel to Story = +30-50% Reel reach
- Comments matter most — reply to every comment with a question to extend thread

**Hook patterns (Reels)**:
- "POV: you just discovered..." (curiosity)
- "Things I wish I knew before..." (regret/loss-aversion)
- "Stop doing X. Do Y instead." (contrarian)
- "Wait for it..." (loop bait — only if payoff is real)
- "X out of 10..." (rating/list)
- Static-text-on-pause-frame opener: bold text on first frame, voice/music starts after 0.3s

**Hook patterns (Carousels)**:
- "I analyzed [N] [thing]. Here's what I found." (data authority)
- "[Number] [things] that [outcome]" (listicle)
- "Save this for [later use case]" (save bait)
- "Read this before you [common action]" (warning frame)

**Monetization gates**:
- Affiliate links in bio: any size
- Branded content / sponsorships: 10k+ followers minimum (but gate is engagement rate, not followers)
- Own product launch: works at any size if niche-aligned, peaks 25k+
- Page sale: 50k+ in monetizable niche, sells $5-50k on platforms like Fameswap

**Tools**: Metricool / Later (schedule), Canva (carousels), CapCut (Reels), Reelsmate / Predis (idea/scrape), AI background music removal for sourced content

### TikTok

**The opportunity**: fastest growth platform — algorithm gives every post a fair shot regardless of follower count. A faceless page can hit 100k followers in 30 days with right niche + 5/day cadence.

**Niche selection**:
- TT eats: storytelling, satisfying/oddly-specific, education-with-personality, drama/recap, comedy, food, beauty, tech-toys, compilation, ASMR
- TT struggles with: B2B, polished corporate, long-form thoughtful

**Format**:
- **Vertical 9:16**, 7-30s for fastest growth, up to 3min if hook holds
- Captions ON SCREEN are mandatory (60%+ watch sound-off in feed)
- First 1.5s = make-or-break

**Cadence**: 3-5/day in growth phase. Yes, every day. Quality matters less than quantity once hook+niche are dialed.

**Growth mechanic**:
- Algorithm tests new posts on small audience batches → expands if retention > niche threshold
- The number that matters: **average watch time as % of video length** (>70% = explosion)
- Comments and shares > likes
- Reply-to-comment with video = free distribution + extends content series

**Hook patterns**:
- "I just found out that..." (discovery)
- Direct question to camera: "Did you know..."
- Mid-action open: drop into the moment, no setup
- Pattern interrupt visual: weird object, color, motion in frame 1
- Text-only on solid color frame for first 0.3s ("This is illegal in 12 countries...")

**Avoid**:
- Slow intros, logos, "hi guys"
- Watermarks from other platforms (TT down-ranks reposts with TikTok→IG→TikTok loops)
- Banned terms (depends on niche — health/finance has long blocklist)

**Monetization gates**:
- Creator Fund / Creator Rewards: 10k followers + 100k views/30d. Pays poorly ($0.02-0.04/1k).
- TikTok Shop affiliate: 1k followers, very high earning potential in commerce niches (US/UK live)
- Brand deals: 50k+, varies wildly by niche (beauty/fashion >> personal finance for TT)
- Page sale: 50k-1M monetizable niche, $1-30k typical

**Tools**: CapCut (native, free, best mobile editor), TikTok native scheduler, ViralFindr / Pentos (competitor tracking), Splice (B-roll)

### Twitter / X

**The opportunity**: highest leverage for B2B, tech, finance, niche-tactical content. Lowest production cost (text). Best at converting attention → off-platform action (newsletter, product, lead).

**Niche selection**:
- X eats: tech, AI, founders/business, finance/investing, crypto, politics (use carefully), philosophy, writing, sports analysis
- X struggles with: visual lifestyle (IG wins), entertainment compilations (TT wins)

**Format**:
- **Single tweets**: 6-10/day. Punchy. Standalone insights.
- **Threads**: 2-5/week. 5-15 tweets. Hook tweet + payoff structure.
- **Image tweets**: screenshot of long text (the "screenshot longform" trick) — bypasses character limit, drives saves
- **Reply game**: 50+ thoughtful replies/day in your niche feed = 80% of growth in first 90 days

**Cadence**: 8-15 posts/day across tweets + replies + retweets-with-comment. X rewards volume more than any other platform.

**Hook patterns (single tweets)**:
- "[Counterintuitive truth]." (1 line, period)
- "If I could [action] over again, I'd [thing nobody does]."
- "[N] [thing] that [outcome]:\n\n1. ..." (mini-thread style single tweet)
- "I spent [time/money] on X so you don't have to. Here's what I learned:" (threadable)

**Thread structure**:
- T1 (hook): result + promise. "I went from $0 to $X in N months. Here's the playbook:"
- T2-N: one insight per tweet, numbered, scannable
- Final tweet: CTA — RT + follow + link to longer resource
- Pin best-performing thread to profile

**Growth mechanic**:
- Replies to bigger accounts in your niche = primary growth lever
- Algorithm boosts tweets that get replies > likes (replies = effort = signal)
- Avoid external links in tweet body (kills reach) — drop links in reply or "link in bio"

**Monetization gates**:
- X Premium ad revenue share: must be Premium subscriber + 500 followers + 5M impressions/3mo
- Newsletter conversion: top-of-funnel tactic — drive to Substack/Beehiiv from tweet replies
- Affiliate / digital product: works at any size with right niche (IM, fitness, software)
- Sponsored tweets: 10k+ in commercial niche

**Tools**: Hypefury / Typefully (scheduling + thread tools + analytics), Tweet Hunter (idea mining + AI), Twemex (sidebar — best tweets from any account)

### Bluesky

**The opportunity**: early-mover advantage. ~20M users (growing), engagement rate is 5-10× X right now because feed isn't algorithmically suppressed. Niche communities forming. Best timing window 2025-2026.

**Niche selection**:
- BS eats: tech, science, art, indie creators, journalism, queer/leftist communities, niche academia, anti-corporate, longform thinkers
- BS struggles with: pure marketing/grift content (community pushes back), generic motivational

**Format**:
- Short posts (300 char limit), threaded posts, image/GIF posts
- Custom feeds (algorithmic feeds users subscribe to) — getting into popular niche feeds = primary growth lever
- Starter Packs (curated user lists) — get added to packs in your niche

**Cadence**: 3-8 posts/day. Lower volume than X. More conversation, less broadcast.

**Hook patterns**:
- More earnest than X — drop the "guru" voice
- "I've been thinking about X..." (genuine inquiry > bold claim)
- Niche-tactical observations (the algorithm hasn't been gamed yet, real value wins)
- Tech/science/art deep dives outperform business motivation

**Growth mechanic**:
- Reply to mid-tier accounts (1k-10k followers) — community is small, replies get noticed
- Get pinned in custom feeds for your niche
- Cross-post X threads (Bluesky audience overlaps but isn't identical)
- Don't auto-cross-post from X — community resents it

**Monetization gates**:
- No native ad revenue yet
- Newsletter / product CTA only
- Pre-monetization phase = build audience cheap, monetize when/if Bluesky adds revenue features

**Tools**: Native scheduler limited; use Buffer (added BS support 2024), Graze (custom feed builder), SkyFeed

### Facebook

**The opportunity**: under-rated for theme pages. Older demographic, high purchase intent. Reels reach is good (FB pushes Reels hard to compete with IG/TT). Groups are massive distribution channels.

**Niche selection**:
- FB eats: age 35+ niches (parenting, midlife health, retirement, hobbies — gardening, fishing, woodworking, RV, classic cars), local/regional, religious/spiritual, recipe/food
- FB struggles with: Gen-Z trends, edgy comedy, fast-cycle tech

**Format mix**:
- **Reels**: same content as IG Reels — cross-post free. FB Reels reach is currently strong.
- **Image posts**: still work on FB (don't on IG). Quote graphics, infographics.
- **Group posts**: own a niche FB Group + post to it = direct distribution to engaged audience
- **Long-form text**: FB users still read long captions, unlike IG

**Cadence**: 1-3/day on page. Group activity = unlimited (more = better engagement).

**Growth mechanic**:
- **FB Group ownership** is the lever. Build a 10k-100k Group around your niche → page is the "front door", group is the community → cross-promote.
- Cross-post Reels from IG via Meta Business Suite (free, same content, separate analytics)
- FB Page audience skews 35-65 — match content to that demographic

**Monetization gates**:
- In-Stream ads (long video): 10k followers + 600k watch min/60d
- Reels Play bonus (intermittent program — check eligibility)
- Stars (live streaming tips): low payout
- Lead gen via Group → email list = the highest-value monetization (more than ads)
- Affiliate / product: works at any size in commerce niches

**Tools**: Meta Business Suite (free scheduling for FB+IG), Group Insights (built-in)

### Pinterest

**The opportunity**: search engine that looks like a social platform. Pins last for **months/years** (vs. hours on TT/IG). Buyer intent is 2-3× higher than any other platform. Drives blog/store traffic at low CAC.

**Niche selection**:
- Pinterest eats: home decor, DIY, recipes, fashion, beauty, weddings, travel, parenting, gardening, finance/budgeting, education printables, planner/productivity, women-skewed content (60%+ female user base)
- Pinterest struggles with: B2B, news, edgy/controversial, gaming, crypto

**Format**:
- **Standard Pins (vertical image)**: 1000×1500px, bold overlaid text, click-through to your blog/store/landing page
- **Idea Pins (multi-page)**: native content, 5-20 pages — like a Story but evergreen
- **Video Pins**: 6-15s, vertical, top of feed
- **Rich Pins**: pull metadata from your site (product/recipe/article) — set up via domain claim

**Cadence**: 5-15 fresh pins/day in growth phase. Pinterest rewards fresh URLs + fresh visuals (not pin reposts).

**Pin design rules**:
- Text overlay is mandatory ("How to...", "10 ways to...", "DIY [thing] under $X")
- Bright colors, faces if relevant, clear hierarchy
- Brand consistency across pins (same fonts, palette) = looks pro = saved more

**Growth mechanic**:
- **SEO**, not algorithm. Pin titles + descriptions + board names = keywords.
- Use Pinterest Trends tool — find seasonal/evergreen keywords
- Each blog post = 5-10 different pin variants (different headlines, different visuals)
- Group boards (joining established niche boards) accelerated growth historically — less effective now but still useful

**Monetization gates**:
- No native ad rev share for creators (yet)
- Affiliate links direct on pins: ALLOWED (since 2020) — biggest theme-page money on Pinterest
- Drive to: blog (display ads + affiliate), Etsy/Shopify store, lead magnet → email list
- Pinterest is the #1 platform for **passive blog traffic** + **Etsy/digital product sales**

**Tools**: Tailwind (schedule + Tribes + analytics), Canva (pin design), Pin Inspector (competitor analysis), Pinterest Trends (free), Pinterest Predicts annual report (plan year ahead)

## The content engine (cross-platform pipeline)

```
[1. SOURCE]
  ├─ Public-domain / CC footage (Pexels, Pixabay, Storyblocks)
  ├─ Re-edit existing viral content with permission/credit
  ├─ AI-generated (Pictory, Invideo, Sora when available, ElevenLabs voice)
  ├─ Original B-roll (commission $5-50/clip on Fiverr or shoot once-monthly)
  └─ Curate from your own product/service if applicable
       ↓
[2. EDIT — produce ONE master unit per idea]
  ├─ 9:16 vertical (TT, IG Reels, YT Shorts, FB Reels)
  ├─ 16:9 horizontal (YT long-form)
  ├─ 1:1 square (FB feed, IG carousels)
  └─ 1000×1500 vertical static (Pinterest pin)
       ↓
[3. ADAPT per platform]
  ├─ Captions (each platform has own copywriting style)
  ├─ Hashtags (research per platform — IG 3-5, TT 3-5, YT 0-3, X 0-1, BS 1-3, FB 1-3, Pinterest keywords>hashtags)
  ├─ Hook tweak (TT: visual-first; YT: title+thumb; Pinterest: text overlay)
  └─ CTA per platform (IG: comment; TT: comment; YT: subscribe; X: RT/follow; FB: share/group; Pinterest: click)
       ↓
[4. SCHEDULE]
  └─ Metricool / Buffer / Publer — multi-platform
       ↓
[5. ENGAGE]
  └─ Reply to every comment in first 60min (algorithm boost)
       ↓
[6. ANALYZE WEEKLY]
  ├─ Top 3 by reach (why) — make 2 more like it
  ├─ Bottom 3 (why) — kill the format
  └─ Best-performing hook patterns → feed back into idea pipeline
```

## Niche selection framework

Before opening ANY page, validate the niche on 5 axes:

| Axis | Question | Pass threshold |
|---|---|---|
| **Search volume** | Does the topic have 50k+ monthly searches (Pinterest/YT) or 10k+ TT hashtag views/day? | Must pass |
| **Existing creators** | Are there 3+ pages with 100k+ followers in this exact sub-niche? | Yes = validated demand. Zero = either you're early or there's no market. |
| **Monetization path** | Can you name 3 affiliate programs / products / lead magnets that match? | Must pass |
| **Content sustainability** | Can you produce 100 posts on this niche without repeating? | Must pass |
| **Personal interest** | Will you still care in 12 months? | Soft requirement (outsourcing later mitigates) |

Niches that consistently pass for theme pages:
- Personal finance subniches (FIRE, dividend investing, side hustles for [demographic])
- Fitness subniches (calisthenics, mobility for desk workers, postpartum, over-50)
- Pet subniches by breed
- Hyper-specific food (one-pan, air fryer, low-FODMAP, ADHD-meals)
- Aesthetic niches (dark academia, cottagecore, minimalist apartments)
- Hobby gear (hiking, mechanical keyboards, fountain pens, vinyl)
- Career/skill (copywriting, AI for [profession], freelance for [niche])
- Education shorts (history, science explained, language learning)

## Process (when invoked)

### Step 1: Define the page
Ask user (or read from `references/my-page-roster.md`):
- Niche (must be specific — challenge if too broad)
- Goal: ad revenue / affiliate / own product / lead magnet / sale
- Time budget per week
- Production capacity (solo, with editor, fully outsourced)
- Existing IP (own footage? brand? product?)

### Step 2: Validate niche
Run the 5-axis framework above. If 2+ fail, suggest pivot or sub-niche.

### Step 3: Pick platforms
Default platform priority (start with 1-2, expand as system runs):

| Goal | Primary | Secondary |
|---|---|---|
| Fastest growth | TikTok | Instagram |
| Highest revenue/view | YouTube | (Pinterest if niche fits) |
| Buyer intent / store traffic | Pinterest | Instagram |
| B2B / tactical / monetize fast | Twitter/X | LinkedIn (covered separately in social-content) |
| Older demo / community | Facebook + Group | Pinterest |
| Early-mover advantage | Bluesky | Twitter/X |

Don't try to launch all 7 at once. Pick 1 primary + 1 secondary. Add platforms only after primary hits 10k followers.

### Step 4: Build the content pipeline
- Define sourcing strategy (which of the 5 source types — see content engine)
- Set up tooling (CapCut + Metricool minimum)
- Create platform-specific templates (overlay style, hashtag bank, hook bank)
- Set cadence calendar

### Step 5: Set monetization gate
- Document the threshold metric for this page (e.g. "10k IG followers → launch lead magnet for ConvertKit affiliate")
- Pre-write the monetization path so it's ready when you hit the gate (not improvised)

### Step 6: Output
Write to `state/theme-page/<page_id>/`:
- `page-spec.md` — niche, goal, platforms, cadence, monetization gate
- `content-pipeline.md` — sourcing + editing + scheduling system
- `hook-bank.md` — 30+ hooks scraped from competitors, organized by pattern
- `posting-calendar.md` — 30-day plan with specific post types per day
- `kpi-targets.md` — 30/60/90 day follower + reach + monetization targets
- `competitor-watch.md` — 10 pages to monitor weekly
