# Owner's Scaling Tiers (verbatim)

## Vertical scaling

| Phase | Daily budget | Step | Cooldown |
|---|---|---|---|
| 1 | under $100/day | +30% | every 48–72h |
| 2 | $100 → $500/day | +20–30% | every 2–3 days |
| 3 | $500+/day | +10–20% | every 2–3 days |

## Horizontal scaling

1. **Duplicate winning ad set into same campaign** — keeps learning context
2. **Duplicate into new campaign** — tests fresh delivery
3. **Add new creatives**

## Hard rule

> Scaling usually fails because of creatives. **You scale creatives that work.**

## Loop

Launch creatives → find winners → scale them → they fatigue → create variations → repeat.
