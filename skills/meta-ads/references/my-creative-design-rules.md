# Owner's Creative Design Rules (verbatim)

## Universal design rules

- Button clear and nice contrast
- Large enough buttons
- Clear calls to action
- Centered calls to action
- Simplicity
- Clear
- Minimal copy
- Clear headlines and main points

## Format constraints (Meta)

- **Headline:** under 40 chars
- **Primary text:** under 125 chars + 4 benefits
- **CTA:** "Shop Now" only
- **Format:** single image or video

## Creative process flow

1. One good sentence + 4 benefits in primary text
2. Strong headline (≤40 chars)
3. Click 3 dots → duplicate → change creative text + media to spawn variants
4. Cap of 3 ads per ad set (open new ad set if more needed)

## Refresh recipe (when an ad fatigues)

From 1 winning ad:
- **3 new hooks**
- **2 new angles**
- **1 new CTA**

Tweak via hook-voice change or other parameter to "relive" the video.

If 4 days post-tweak still unprofitable → close campaign, redo with new creatives.

## Brand model rule

**Highly use our brand-locked models. Don't switch models all the time** if it's UGC. Consistency builds recognition.
