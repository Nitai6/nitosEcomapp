# Owner's SEO Audit Checklist (verbatim)

Every 5 days. SEMrush level. Technical + On-page + Off-page.

## Performance & Core Web Vitals
- Largest Contentful Paint (LCP)
- Interaction to Next Paint (INP)
- Cumulative Layout Shift (CLS)
- Time to First Byte (TTFB)
- First Contentful Paint (FCP)
- Caching Strategies (Browser, Server-side, Edge/CDN, Object, Redis/Memcached)
- Minification (HTML, CSS, JS)
- Compression (Gzip, Brotli)
- Image Optimization (WebP, AVIF, Lazy Loading, Responsive Images/Srcset, Decoding attributes)
- Resource Hints (Preload, Prefetch, Preconnect, DNS-Prefetch, Prerender)
- Critical Rendering Path Optimization
- CSS/JS Deferral and Asynchronous Execution
- Third-Party Script & Pixel Management

## Mobile-Friendliness
- Responsive Web Design
- Viewport Meta Tag Configuration
- Touch Target Sizing & Spacing
- Font Scaling & Interstitial Guidelines
- Desktop/Mobile Content Parity

## Crawlability & Architecture
- XML Sitemaps (Index, Image, Video, News, Mobile)
- HTML Sitemaps
- robots.txt (Allow, Disallow, User-agent, Crawl-delay, Sitemap directives)
- Site Architecture (Flat vs. Deep, Taxonomy, Silo Structure)
- Click-Depth / Crawl Depth Optimization
- URL Structure (Subfolders, Subdomains, Absolute vs. Relative URLs)
- URL Parameters & Dynamic URLs Handling
- Trailing Slashes & Case Sensitivity Consistency
- Faceted Navigation Control
- Orphan Pages Identification
- Crawl Budget Optimization
- Log File Analysis

## Structured Data & Semantics
- Schema.org Vocabularies (Product, Article, FAQPage, LocalBusiness, BreadcrumbList, Organization, Person, VideoObject, etc.)
- Markup Formats (JSON-LD, Microdata, RDFa)
- Entity Association (@id, SameAs)
- Semantic HTML5 Tags (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`)

## Keyword Strategy & Mapping
- Primary/Seed Keyword Targeting
- Secondary/Long-Tail Keyword Integration
- Search Intent Alignment (Informational, Navigational, Transactional, Commercial)
- Keyword Density & Prominence
- TF-IDF Analysis & Semantic Equivalents (LSI Keywords)
- H1 Tag (Single use, Primary Keyword, Exact Match or Close Variant)
- H2 Tags (Subtopics, Secondary Keywords, Question Formatting)
- H3–H6 Tags (Nested Sub-sections, Granular Structuring)

## Content Quality & Depth
- E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness)
- YMYL (Your Money or Your Life) Alignment
- Content Comprehensiveness vs. User Intent
- Freshness Updates & Content Decay Management
- Original Data / Unique Value Proposition
- Avoidance of Keyword Cannibalization
- Avoidance of Thin Content / Duplicate Content

## User Experience (UX) & Formatting
- Readability (Flesch-Kincaid)
- Paragraph Length & White Space (F-Pattern, Z-Pattern)
- Bullet Points & Numbered Lists
- Jump Links / Table of Contents
- Above-the-Fold Optimization

## Internal & External Linking
- Contextual Internal Links
- Anchor Text Optimization (Exact, Partial, Generic, Branded)
- Link Equity Distribution (PageRank Flow)
- Outbound Links to High-Authority/Trust Sources

## Image & Media Optimization
- Alt Text Optimization (Descriptive, Keyword-Aware, Accessibility)
- Image File Names (Dash-separated, Keyword-rich)
- Image Captions & Surrounding Text Context
- Video Embed Context & Transcripts
