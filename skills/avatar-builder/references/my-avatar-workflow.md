# Owner's Avatar Workflow (verbatim)

## 8 steps

1. **Rough avatar idea** — simple concept. Examples: "23-year-old fitness influencer", "luxury lifestyle influencer", "travel vlogger", "gamer girl streamer".

2. **ChatGPT prompt expansion** — turn the rough idea into a detailed higgsfield-image (GPT-image 2.0 route) prompt:
   > "Turn this idea into a detailed higgsfield-image (GPT-image 2.0 route) prompt for generating a realistic AI influencer. Idea: [X]"

3. **higgsfield-image generation** — open the higgsfield-image Playwright wrapper (GPT-image 2.0 route) and run the expanded prompt.

4. **Pick + download reference** — review outputs, pick the **most realistic** image, download. This is the **reference image** for your influencer.

5. **Lock character identity** using the reference image.

6. **Generate new scenes** with the same avatar.

7. **Build a content library**.

8. **Turn images into video** — once the avatar exists, generate unlimited content with the same digital person.
