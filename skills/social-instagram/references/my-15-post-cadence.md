# 15-Post Weekly Cadence — Owner Spec

## Targets

| Format | Weekly count | % of feed |
|---|---|---|
| Carousels | 9–10 | 60–70% |
| Reels | 3 | ~20% |
| Static | 3 | ~20% |
| Stories | 8 | n/a (not feed) |

## Why carousels dominate
Owner brief: carousels drive saves; saves are the strongest brand-affinity signal IG's algorithm rewards. Reach = saves × 3 + comments × 2 + likes (in scoring rubric).

## Daily slot map (default)

| Day | Slot 1 | Slot 2 | Story |
|---|---|---|---|
| Mon | Carousel 11:00 | Reel 17:00 | BTS |
| Tue | Carousel 11:00 | Static 17:00 | Poll |
| Wed | Carousel 11:00 | Carousel 17:00 | Product use |
| Thu | Reel 11:00 | Carousel 17:00 | DM screenshot proof |
| Fri | Carousel 11:00 | Static 17:00 | Founder takeover |
| Sat | Static 11:00 | Carousel 17:00 | BTS |
| Sun | Carousel 11:00 (light day — 1 post) | — | Recap |

## Slot → seed mapping

| Format | Best seed type |
|---|---|
| Carousel | Info-dense educational, "X mistakes", "How to Y", framework |
| Reel | Trend remix, product-shot, founder POV |
| Static | Quote, proof screenshot, before/after, social proof |

## Stories pillar mix (8/week)

- 2 BTS (workspace, packaging, day-in-life)
- 2 poll/quiz (engagement bait)
- 2 product use (lifestyle)
- 1 customer DM screenshot proof
- 1 founder story-takeover

## Superprofile funnel attach rule

A calendar slot can set `attach_funnel: true` — typically:
- Reels with strong educational hook → funnel for related freebie
- Carousels offering a checklist → funnel for full-version download

Funnel deploys via separate `dm-funnel` agent.
