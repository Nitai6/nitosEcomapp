# Owner's Banner Formulas (verbatim)

## Banner image formula

```
[Person] + [Outcome] + [Emotion]
```

Must be **positive, exciting**.

## Banner text formula

```
[BENEFIT] = [DESIRED RESULT] + [LOWER EFFORT / LOWER COST / LOWER RISK OF PAIN POINT]
```

### Thumb rules for banner text

- 6–12 words max
- Focuses on **outcome, not process**
- Sounds good when read fast
- Emotional, not technical
- No commas if possible

## CTAs

- **Primary CTA** = more precise & specific
- **Secondary CTA** = broader

## Subheadline formula

```
[Simple Mechanism] + [Key Benefit]
```

## Validator

Before shipping a banner, the agent runs:
1. Word count 6–12 ✓
2. No commas ✓ (warn if any)
3. Outcome-language test (verb of result, not verb of process) ✓
4. Mechanism present in subheadline ✓
5. Primary CTA more specific than secondary ✓
